# Sprint 01 (Week 1) — Execution Plan

## Objective
تجهيز الأساس التشغيلي: الصلاحيات + البيانات الأساسية + تجهيز هيكل قاعدة البيانات الأولي.

## Scope
1) WBS-01: إدارة المستخدمين والصلاحيات
2) WBS-02: البيانات الأساسية
3) إعداد مخطط قاعدة بيانات ابتدائي للوحدات الثمانية

## Deliverables
- Permission Matrix (module/action based)
- Role Definitions + Approval Authorities
- Master Data Dictionary
- SQL Schema v0.1 (core tables + keys)
- Seed data template

## Tasks
- [ ] تعريف الأدوار الرسمية وتفاصيل كل صلاحية
- [ ] تعريف قواعد الرؤية حسب القسم
- [ ] توحيد الحقول الإلزامية لكل كيان رئيسي
- [ ] بناء SQL DDL مبدئي للجداول الأساسية
- [ ] ربط جداول المستخدمين/الأدوار/الصلاحيات بالموافقات

## Acceptance for Sprint 01
- يمكن إنشاء مستخدم وربطه بدور وقسم
- يمكن تطبيق صلاحية CRUD + Approve على مستوى الوحدة
- جداول البيانات الأساسية جاهزة للاستخدام
- مخطط قاعدة البيانات مهيأ للانتقال إلى Sprint 02

## Risks
- غياب ترميز موحد للمواد/المستودعات/العملاء
- تعارضات صلاحيات موافقات بين الإدارات

## Mitigation
- اعتماد أكواد موحدة من البداية
- مصفوفة صلاحيات/موافقات واضحة وموقعة قبل تنفيذ الواجهات
